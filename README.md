## Calories Calculator 
This app uses complex formulas such as Mifflin-St Jeor's equation for both male and female. 
Also, this app is helpful for transgender women/transgender men who are currently on HRT 
from 0-12 months for trans-men, and 0-24 months for trans-women.

## Tech used
- HTML
- CSS
- JavaScript
